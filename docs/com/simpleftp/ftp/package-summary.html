Package com.simpleftp.ftp

  • Class Summary 
    Class Description
    FTPConnection
    This is the main class used for provided a client connection to a FTP server It provides a subset of the features provided by org.apache.commons.net.ftp.FTPClient This class also abstracts some calls of that library to hide any unnecessary details It also doesn't include any functions that are not required by the system so not to complicate things The library also needs to be wrapped so that exceptions for this system can be thrown and certain constraints to be put on the calls Also required to provide logging
    FTPConnectionDetails
    This class provides details to be used in creating a FTPConnection Encapsulates all these details into one place
    FTPPathStats
    This class returns a collection of statistics for a specified path in one location
